# bug-008: ヒートマップの高密度領域が低密度領域に上書きされる

## Status: Open

## 概要

カバレッジヒートマップで、新しいキャプチャ追加後に既存の赤色（高密度）領域が白系の薄い色に変わってしまう。ヒートマップの色が「どこにコーナーが多いか」を正しく表現しなくなり、「次にボードをどこに置くべきか」の判断ができない。

## 再現手順

1. Calibrationタブでカメラを選択しライブビューを開始
2. 画像の左半分でキャプチャを数回取得 → 左半分が赤色になる
3. 画像の右半分でキャプチャを1回取得 → 左半分の赤色が薄い色に変わる

## 期待する動作

キャプチャ追加後も、既存の高密度領域は赤色のまま維持される。新しい領域は低密度として青〜緑で表示される。
